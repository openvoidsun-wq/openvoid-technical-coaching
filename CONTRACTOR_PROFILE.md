# Contractor Profile

Operating name: OpenVoid

Email: openvoidsun@gmail.com

## Fit

Overflow technical coaching for tutoring companies, bootcamps, mentor networks, and small teams that need learners unblocked without creating academic-integrity risk.

Best-fit sessions:

- Python debugging and beginner-to-intermediate coaching.
- SQL query help and database fundamentals.
- JavaScript/web project troubleshooting.
- Excel, data, and automation workflow support.
- AI workflow coaching for study and small-business operators.
- AP CS, university, bootcamp, adult reskill, and professional upskilling support.

## How Sessions Work

1. Learner or team brings a concrete blocker.
2. We diagnose the issue live by screen share.
3. The fix is explained step by step.
4. A concise recap and next-step checklist is provided after the session.

## Paid Pilot

- USD 35 / 45 minutes.
- One learner scenario or internal sample case.
- Prepaid, escrow, platform hourly contract, or confirmed transfer before work starts.
- No unpaid trial requirement.

## Boundaries

Accepted:

- tutoring,
- debugging,
- code review,
- project rescue with explanation,
- learner-facing recap notes,
- contractor-style overflow work.

Not accepted:

- live exam help,
- account impersonation,
- hidden completion of graded submissions,
- pay-after-grade/result arrangements.
