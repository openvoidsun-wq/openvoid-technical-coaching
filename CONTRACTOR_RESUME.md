- Python: debugging, beginner-to-intermediate concepts, scripts, APIs, automation

## Delivery Style

- Live screen-share coaching.
- Diagnose the blocker first.
- Explain the failure point in plain language.
- Guide the learner through the fix without impersonation.
- Send a short written recap with next steps.

## Integrity Boundary

Accepted:

- tutoring,
- debugging,
- code review,
- concept explanation,
- project rescue with learning value,
- recap notes.

Not accepted:

- hidden completion of graded submissions,
- pay-after-grade/result arrangements,
